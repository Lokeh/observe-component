# List example

To run (using yarn):
```bash
git clone <observe-component repo>
yarn
yarn run build
cd observe-component/examples/list
yarn
yarn start
```

Or, using npm:
```bash
git clone <observe-component repo>
npm install
npm run build
cd observe-component/examples/list
npm install
npm start
```

Then direct your browser to [http://localhost:8080/webpack-dev-server/](http://localhost:8080/webpack-dev-server/)
